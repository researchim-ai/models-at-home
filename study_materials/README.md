# Study materials (учебные материалы)

Materials are provided in two languages. The app loads the version that matches the current interface language.

- **`en/`** — English (LLM.md, VLM.md, transformer.md)
- **`ru/`** — Русский (LLM.md, VLM.md, transformer.md)

Internal links between documents (e.g. from VLM.md to LLM.md) work in the Study Center: clicking them opens the corresponding document in the same language.
